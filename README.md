# 🎮 Game Prototyping Project

A single-player puzzle game that explores the theme of **contrast** by allowing the player to switch between different time periods to navigate the environment.

---

## 🧠 Concept

- Solve environmental puzzles to progress through each level  
- Use a **time travel ability** to change locations across time  
- Switching time periods reveals new paths, removes obstacles, or creates new ones  
- Single-player experience focused on problem-solving and exploration
